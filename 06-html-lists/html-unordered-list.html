Unordred List with <ul> & <li>

HTML lists allow web developers to group a set of related items in lists.

<ul> element

The HTML <ul> element is for grouping a collection of items that do not have a numerical ordering.
Typically, unordered-list items are displayed with a bullet, which can be of several forms, like a dot, a circle, or a square.

Unordered list are mainly important in navigation to show menu items. 

        <ul>
            <li>item 1</li>
            <li>item 2</li>
        </ul>

list-style-type

The list-style-type CSS property sets the marker of a list item element.
The list-style-type property may be defined as any one of: symbol, string, custom-indent or none 

        <ul style="list-style-type: square">
            <li>item 1</li>
            <li>item 1</li>
        </ul>
