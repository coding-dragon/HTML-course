<a> element styling

We use anchor element most of types in document to jump from one document to another. Its good practice to represent link in attentive style

styling link as button

To remove underline from link, text-decoration: none is used in css styling. 

        #myBtn {
            background-color: #3124ee;
            color: #fff;
            padding: 10px 20px;
            text-decoration: none;
            border: 4px solid #9e98f1;
            border-radius: 5px;
        }

        <a href="https://www.google.com" id="myBtn" >Click Here</a>
